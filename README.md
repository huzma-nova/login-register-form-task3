# 🔐 Login & Registration Form

A modern and interactive **Login & Registration System** built using **HTML, CSS, and JavaScript**. This project demonstrates basic frontend development concepts like form validation, DOM manipulation, and UI interactivity.

---

## 📌 Overview

This project provides a simple authentication interface with:

* Login functionality
* User registration form
* Real-time validation and error handling
* Smooth UI switching between forms

---

## ✨ Features

✔ Toggle between Login & Register forms
✔ Input validation with error messages
✔ Password show/hide functionality 👁️
✔ Clean and modern gradient UI
✔ Success message display (no alert boxes)
✔ Beginner-friendly and responsive layout

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript
